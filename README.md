# TodoApp
This is application has developed in Node js (Express js).


Technologies used in project listed down.
- HTML 5
- CSS 3
- jQuery
- Ajax
- Bootstrap 4
- Node js
- Express js
- MongoDB

# To run

Follwo's Steps.
- Clone repository.
- Get into TodoApp directory.
- Then run "npm install".
- After that run "node app".
- Go to "http://localhost:3000/".
